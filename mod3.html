Workshop Introduction

  • Name is Working and Deploying Websites from Scratch
  • This Workshop is targeted to Total Beginners to HTML and Web design
  • The workshop is meant to be from 1.5hrs to 2 hrs

Learning Objectives

  • Learners can understand how to build and deploy a website from Scratch
  • Learners will get to know the basics of HTML and CSS
  • Learners will get Insights on Frontend Web Development and they will get to know where to apply these Insights

Assessments

  • Leaners can experiment with and implement each thing they learned that day and when they apply these to a final projet of the day,Its a win.
  • When they can create small webpages with some basic links and tags,its a win.
  • Deploying the thngs they leant in perfect situations can be a criteria for Success.
